E-FARM Token

EFARM token is the in-game currency. It will be used for trading, exchanging, improving and upgrading Assets special qualities, in the first phase.

About E-FARM

Farm Earner is a web-based single player game application that is built on the Binance Smart Chain Network. Our players/users will be able to access the NFTs Marketplace to buy from daily game characters to some future rare assets and play the web based game. E-farm will be the main game currency which can be exchanged on Defi Crypto Exchange and it will use for transactions in the Farm Earner Ecosystem. Players will be able to acquire most games assets, Farm Lands and beautiful collections of farm animals using E-Farm.
